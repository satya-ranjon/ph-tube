1 - Discuss the scope of var, let, and const

What is Scope

Scope basically refers to the area in which these variables can be used.

- Var

Var declarations can be function/locally or globally scoped.

Global : When a var variable is declared outside of a function, the scope is global. This means that any variable declared outside of a function block using the var keyword is accessible throughout the entire window. 

var num = 20; 
function myFun() {
    console.log(num); 
}

Function/Local : When a variable is declared inside a function, it has function scope. This indicates that it is accessible and limited to that function. 

function myFun() {
    var num = 10;
}

- Let

let has block scope

Block Scope : Anything inside the {} is considered a block. Block scope means if “num” variable is declared inside the block then it is not accessible outside the block . 

function myFun() {
    let num = 20;
} 
console.log(num)

- Output

ReferenceError: a is not defined

* Const can be updated, but cannot be redeclared.

- Const

Const also has block scope like let

Block Scope : Anything inside the {} is considered a block. Block scope means if “num” variable is declared inside the block then it is not accessible outside the block . 

function myFun() {
    const num = 20;
} 
console.log(num)

- Output

ReferenceError: a is not defined

* Const cannot be updated and redeclared.

2 - Tell us the use cases of null and undefined

null is used to represent an explicitly empty or non-existent value. undefined is used by JavaScript itself to represent the absence of a value, such as when a variable is declared but not initialized.

- null

Explicitly empty value : null is used to represent an explicitly empty or non-existent value. 

let str = null;

Resetting the value: When resetting the initial data 

let obj = {name:"one"};
obj = null;

- undefined

Default Value : When the variable is declared but not set to a value, the undefined value is automatically assigned.It is often used as the default initial value for function parameters or variables when their value is not specified 

function myFun(name) {
console.log(`Hello, ${name}`);
}
myFun();

- Output

Hello, undefined

Property Not Found : When you try to access an object property that doesn't exist, JavaScript returns undefined. 

const user = { name: 'Alice' };
console.log(user.phone);

- Output

undefined

Array Elements : When you access an array element beyond its length, you get undefined. 

const newArr = [1, 2, 3];
console.log(newArr[10]);

- Output

undefined

3 - What do you mean by REST API?

An API, or application programming interface, is a set of rules that define how applications or devices can connect and communicate with each other. A REST API is an API that conforms to the design principles of REST, or the representational state transfer architectural style.
